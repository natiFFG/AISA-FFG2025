# Donation Platform - AISA

## Description
This repository contains the source code and documentation for the donation platform web application developed for AISA, aligned with their mission to attract and retain donors. The platform facilitates secure donation management, improves transparency, and enhances donor engagement through features like donation tracking, customizable contributions, and gamification elements.

## Key Features
- **User Authentication**: Secure registration and login for donors.
- **Donation Management**: Update payment methods, modify donation amounts, and access donation receipts in PDF format.
- **Informational Sections**: Latest news, program details, and transparent allocation of donations.
